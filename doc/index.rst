ewoksutils |version|
====================

*ewoksutils* provides utilities for `ewoks <https://ewoks.readthedocs.io/>`_ developers.

*ewoksutils* has been developed by the `Software group <http://www.esrf.eu/Instrumentation/software>`_ of the `European Synchrotron <https://www.esrf.eu/>`_.
